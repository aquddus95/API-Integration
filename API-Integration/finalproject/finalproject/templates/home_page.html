Find the Perfect Restaurant and Recipe Combo


FoodSpec is an amazing application that not only gives you detailed information regarding a restaraunt that you may search but also provides recommended recipes based of the restaurant's cusine type. What makes FoodSpec unique is that it scours the likes of Yelp and Zomato to provide you with not only more reviews but also information that one of these sources may include while the other does not. The objective of this application is for the user to not only be able to look up restaurants and get recipe recommendations but to be able to get recipe and restaraunt recommendations based of the user's search history on the website
